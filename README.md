# Next Word Predictor

A project for predicting the next word in a sequence.

## Overview

This repository contains tools and models for next word prediction tasks.

## Getting Started

To get started with this project, clone the repository and explore the code.

```bash
git clone https://github.com/irritatednishant/next_word_predictor.git
cd next_word_predictor
```

## Features

- Next word prediction
- Model training and evaluation
- Text preprocessing utilities
